## 🚀 Getting Started

### Prerequisites

- C++ compiler (clang or g++)
- Text editor or IDE (VS Code recommended)
- Basic command line knowledge

### Quick Start

Each lesson folder contains `.cpp` files that can be compiled and executed individually.

```bash
# Navigate to the project directory
cd "Learn C++"

# Compile a specific file
clang -o output_name filename.cpp

# Run the compiled program
./output_name
```

## 📂 Lesson Structure

### 1. **Getting Started** (`get started/`)

Start your C++ journey with the basics.

- **day01** - Your first C++ program
- **print_number** - Basic output operations

### 2. **Data Types** (`data_type/`)

Learn about different data types and how to use them.

- Integer, Float, Double, Char, String, Boolean

### 3. **Variables** (`variable/`)

Understanding variable declaration, initialization, and scope.

- Variable declaration and assignment
- Constants (`const` keyword)

### 4. **Comments** (`comment/`)

Write meaningful comments in your code.

- Single-line comments (`//`)
- Multi-line comments (`/* */`)

### 5. **User Input** (`user_input/`)

Learn how to take input from users.

- **cin1** - Basic input with cin
- **input** - Storing and using input
- **input_info** - Multiple input values
- **multi_value** - Handling multiple data types

### 6. **Arrays**

Multiple lessons on working with arrays:

#### **Start Array** (`start_array/`)

- **array** - Basic array declaration and access
- **array_string** - Arrays of strings
- **struct** - Basic structure usage
- **struct_array** - Arrays of structures
- **struct_input** - Taking structure input from users

#### **Array** (`array/`)

- **array** - Advanced array operations
- **array_with_string** - Working with string arrays
- **struct** - Structure deep dive
- **struct_array** - Complex structure arrays
- **info** - Array information and manipulation

### 7. **Operators** (`operator/`)

Master different types of operators:

#### Arithmetic Operators

- **arithmetic** - Basic math operations (+, -, \*, /, %)
- **assignment** - Assignment operators (=, +=, -=, etc.)
- **assignment_operators** - Extended assignment operations

#### Relational & Logical Operators

- **relational** - Comparison operators (==, !=, <, >, <=, >=)
- **logical_and_operator** - AND operator (&&)
- **logical_or_operator** - OR operator (||)
- **logical_not_operator** - NOT operator (!)

#### Other Operators

- **inser_ext_oper** - Insert and extract operators
- **if_else_more** - Operators with conditionals
- **exercise** - Operator practice problems

### 8. **Flow Control** (`flow_control/`)

Control program execution flow:

- **if** - Basic if statements
- **if_else** - If-else decision making
- **switch** - Switch case statements
- **more_switch** - Advanced switch usage
- **ternary** - Ternary operator (? :)
- **nest** - Nested conditionals
- **login_with_ifelseif** - Practical login example

### 9. **Loops** (`loop/`)

Repeat code blocks efficiently:

- **for_loop** - For loop basics and iteration
- **while_loop** - While loop implementation
- **while_loop_string** - Loops with string operations
- **do_while_loop** - Do-while loop variations

### 10. **Functions** (`function/`)

Learn to write reusable code with functions:

- **function** - Basic function definition and calling
- **parameters** - Function parameters
- **multiple_par** - Multiple parameters
- **default_par** - Default parameter values
- **real_life_ex** - Practical function examples
- **exercises/** - Function practice problems

### 11. **Object-Oriented Programming (OOP)** (`oop/`)

Master OOP principles:

#### Core Concepts

- **oop** - Introduction to OOP
- **private_keyword** - Access modifiers

#### OOP Pillars

- **Abstraction/** - Data abstraction
- **Encapsulation/** - Bundling data and methods
- **Inheritance/** - Code reuse through inheritance
- **constructor/** - Constructors and initialization

### 12. **Exercises** (`exercises/`)

Practical projects to reinforce learning:

- **calculator** - Build a simple calculator
- **digital_clock** - Create a digital clock display
- **math** - Mathematical operations and games
- **pattern** - Generate text patterns (stars, diamonds, etc.)
- **todo_sms** - Simple to-do list or messaging system

## 💻 How to Use

### Recommended Learning Path:

1. Start with **Getting Started** to understand basic syntax
2. Learn **Data Types** and **Variables**
3. Master **User Input** for interactive programs
4. Study **Operators** and **Flow Control** together
5. Practice **Loops** with different control structures
6. Write **Functions** to organize code
7. Work with **Arrays** and **Structures**
8. Learn **OOP** principles gradually
9. Complete **Exercises** to solidify your knowledge

### Tips:

- Read the code comments carefully
- Compile and run each example
- Modify examples and experiment
- Try to write similar programs from scratch
- Work through exercises progressively

## 🛠️ Compilation

### Using Clang (macOS/Linux)

```bash
clang -o program_name filename.cpp
./program_name
```

### Using G++ (Linux/Windows)

```bash
g++ -o program_name filename.cpp
./program_name
```

### Using VS Code Build Task

You can also use the built-in task in VS Code: `C/C++: clang build active file`
